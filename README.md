# AR-ASL
![](/examples/3.PNG)

## Functionalities
1. Read Image file/ webcam, and show the corresponding ASL stream

## Requirements
1. Pytesseract
2. OpenCV
3. NLTK
4. GoogleTrans
5. tkinter

## How to Run
Run the following command after installing the requirements and follow the instruction in the menupage.

`python text_to_asl.py`
