# AR-ASL

## Functionalities
1. Read Image file/ webcam, and show the corresponding ASL stream

## Requirements
1. Pytesseract
2. OpenCV
3. NLTK
4. GoogleTrans
5. tkinter

## How to Run
Run text_to_asl jupyter notebook after installing the dependencies.
